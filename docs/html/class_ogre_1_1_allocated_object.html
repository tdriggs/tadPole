Detailed Description

template<class Alloc>
class Ogre::AllocatedObject< Alloc >

Superclass for all objects that wish to use custom memory allocators when their new / delete operators are called. Requires a template parameter identifying the memory allocator policy to use (e.g. see StdAllocPolicy).
