Detailed Description

A "standard" allocation policy for use with AllocatedObject and STLAllocator. This is the class that actually does the allocation and deallocation of physical memory, and is what you will want to provide a custom version of if you wish to change how memory is allocated.

This class just delegates to the global malloc/free.
