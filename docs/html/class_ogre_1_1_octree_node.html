Detailed Description

Specialized SceneNode that is customized for working within an Octree. Each node maintains its own bounding box, rather than merging it with all the children.

void Ogre::OctreeNode::_updateBounds ( void  )
protectedvirtual

Internal method for updating the bounds for this OctreeNode.

Remarks
This method determines the bounds solely from the attached objects, not any children. If the node has changed its bounds, it is removed from its current octree, and reinserted into the tree.
