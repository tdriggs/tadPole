Detailed Description

IMPORTANT: Plugins must override default dictionary name! Base class that texture plugins derive from. Any specific requirements that the plugin needs to have defined before texture/material creation must be define using the stringinterface before calling create defined texture... or it will fail, though, it is up to the plugin to report errors to the log file, or raise an exception if need be.
