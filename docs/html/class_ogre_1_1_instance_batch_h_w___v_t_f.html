Detailed Description

Instancing implementation using vertex texture through Vertex Texture Fetch (VTF) and hardware instancing.

See also
BaseInstanceBatchVTF and
InstanceBatchHW

The advantage over TextureVTF technique, is that this implements a basic culling algorithm to avoid useless processing in vertex shader and uses a lot less VRAM and memory bandwidth

Basically it has the benefits of both TextureVTF (skeleton animations) and HWInstancingBasic (lower memory consumption and basic culling) techniques
