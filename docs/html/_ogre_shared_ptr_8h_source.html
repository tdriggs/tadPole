159  /* DO NOT ADD MEMBERS TO THIS CLASS!
160  *
161  * The average Ogre application has *thousands* of them. Every extra
162  * member causes extra memory use in general, and causes extra padding
163  * to be added to a multitude of structures.
164  *
165  * Everything you need to do can be acomplished by creatively working
166  * with the SharedPtrInfo object.
